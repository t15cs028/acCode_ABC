# このリポジトリについて

AtCoder のマイページから確認できるけど、<br>
参加した AtCoder Beginner Contest のソースコードを保管します。<br>
<br>

## ファイルの命名について
+ コンテスト時間内　：　【問題のアルファベット】.cpp
+ コンテスト終了後　：　【問題のアルファベット】\_【AfterContest】.cpp、（解説見た場合）【問題のアルファベット】\_【hint】.cpp
<br>
